# Контракт ответа агента (LocalScript)

## Формат

- Один блок кода в разметке **` ```lua `** … **` ``` `** с **полным** исполняемым скриптом.
- Не оборачивай код в лишний текст после блока, если задача просит только код.

## Содержание

- Не выдумывай функции и модули, которых **нет** в задаче пользователя и в блоке **Retrieved reference** (RAG). Если API описан там — следуй ему буквально.
- Если пользователь не просил файлы или сеть — избегай **`io.*`**, **`os.*`**, **`loadfile`** произвольных путей.

## Повтор при ошибках

- Если в feedback пришли диагностики линтера или sandbox — выведи **исправленный полный** скрипт в новом блоке ` ```lua `, а не дифф.
