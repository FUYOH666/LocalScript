# Lua 5.4 — краткие заметки для генерации кода

## Версия и стиль

- Целевая версия: **Lua 5.4** (`//` целочисленное деление, `utf8` в стандартной библиотеке).
- Предпочитай **`local`** для переменных и функций верхнего уровня, чтобы не засорять глобальное окружение.

## Функции и возвраты

- **Несколько возвращаемых значений:** `return a, b` — идиоматично для ошибок (`nil, "message"`).
- Локальные функции: `local function f(x) ... end`.

## Итерация и таблицы

- **`ipairs(t)`** — последовательная часть массива (1..n).
- **`pairs(t)`** — все пары ключ–значение; порядок не гарантирован для строковых ключей.
- Доступ к несуществующему ключу даёт **`nil`**, не ошибку.

## Строки и Unicode

- Модуль **`utf8`**: `utf8.len`, `utf8.offset`, кодировка UTF-8 по умолчанию в строковых литералах.

## Типичные ошибки (избегать)

- Забытый **`local`** → случайный глобал.
- Смешение **`nil` в таблице** с «дырами» в массиве — `ipairs` остановится раньше.
- Предположение, что **`#t`** длина массива корректна при дырах в индексах.
